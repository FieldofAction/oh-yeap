## Hidden Features (keyboard shortcuts)

```
?   → System Condition (reading/building/working status)
M   → Pattern Lens (overlay model chips on items)
G   → Connections Graph (interactive node diagram)
K   → Translation Key (Hermetic → Field law names, Console only)
Esc → Close any overlay / graph / egg
```
